## ğŸ¥ Case 1: Medical Clinic Management System

This module allows:

- Registration of doctors and patients  
- Appointment scheduling  
- Doctor availability verification  
- Identification of the most requested doctor  

### Main Classes

- **Clinic**
  - Core system class
  - Manages doctors, patients, and appointments
  - Key methods:
    - `addDoctor()`
    - `addPatient()`
    - `addAppointment()`
    - `mostRequestedDoctor()`
    - `doctorSchedule()`

- **Patient**
  - Stores patient information such as name, age, symptoms, health insurance, and blood type
  - Getters and setters
  - Method `hasHealthInsurance()`

- **Doctor**
  - Stores name, specialization, CRM, and availability
  - Appointment counter
  - Methods:
    - `isAvailable()`
    - `incrementAppointmentCount()`

- **Appointment**
  - Connects doctor and patient
  - Stores date, time, and priority
  - Method `possiblePriority()`

## ğŸ½ï¸ Case 3: Restaurant Management System
